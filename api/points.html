Points and Point Clouds¶

A set of useful classes for 2D and 3D points, vectors, planes, point clouds, and normal clouds.

BagOfPoints¶

class core.BagOfPoints(data, frame)¶

Bases: object

The abstract base class for collections of 3D point clouds.

__init__(data, frame)¶

Initialize a BagOfPoints.

Parameters:
  • data (numpy.ndarray of float) – The data with which to initialize the collection. Usually is of shape (dim x #elements).
  • frame (str) – The reference frame in which the collection of primitives resides.
Raises:

ValueError – If data is not a ndarray or frame is not a string.
shape¶

tuple of int – The shape of the collection’s data matrix.

frame¶

str – The name of the reference frame in which the collection’s objects reside.

data¶

numpy.ndarray of float – The collection’s data matrix.

dim¶

int – The number of entries in the data array along the first dimension. By convention, this is the dimension of the elements (usually 3D).

num_points¶

int – The number of entries in the data array along the second dimenstion. By convention, this is the number of elements in the collection.
