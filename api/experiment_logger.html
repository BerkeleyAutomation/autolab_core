Experiment Logging¶

Classes for logging experimental data to files in a well-specified format.

ExperimentLogger¶

class autolab_core.ExperimentLogger(experiment_root_path, experiment_tag=’experiment’, log_to_file=True, sub_experiment_dirs=True)¶

Bases: object

Abstract class for experiment logging.

Experiments are logged to CSV files, which are encapsulated with the CSVModel class.

__init__(experiment_root_path, experiment_tag=’experiment’, log_to_file=True, sub_experiment_dirs=True)¶

Initialize an ExperimentLogger.

Parameters:
  • experiment_root_path (str) – The root directory in which to save experiment files.
  • experiment_tag (str) – The tag to use when prefixing new experiments
  • log_to_file (bool, optional) – Default: True If True will log all logging statements to a log file
  • sub_experiment_dirs (bool, optional) – Defautl: True If True will make sub directories corresponding to generated experiment name
