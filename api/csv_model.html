CSVModel¶

A model for processing CSV files, including reading, writing, and table structuring.

class core.CSVModel(full_filename, headers_types_list, default_entry='')¶

A generic model for CSV file reading and writing.

Parameters:
  • full_filename (str) – The file path to a .csv file.
  • headers_types_list (list of two-tuples str, str) – A list where each item is a tuple of string header for a column and the correspoding data type as a string.
  • default_entry (str) – The default entry for cells in the CSV.
